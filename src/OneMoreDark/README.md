# OneMoreDark

OneMore Dark Theme is a low-contrast theme based on VS Code Dark theme (with same background color `#1E1E1E` ), applicable to Windows Terminal, Notepad++ and so on...

OneMore 暗色主题是一款基于 VS Code 暗色主题的低对比度主题（均基于底色 `#1E1E1E` ），目前适配于 Windows Terminal, Notepad++ 等 APP。

Colors are almost **evenly spread** around the color wheel to be easily distinguished.

采用的颜色间隔基本**均衡**以保证识别度。
